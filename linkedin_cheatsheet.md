## About Section: 3-Paragraph Template

### Template (copy and customize)

```
[HOOK — 2-3 lines, above the fold]
I [what you do] for [who]. With [X years / X projects] in [domains], I've [biggest achievement with a number]. Currently [what you're working on / learning].

[STORY — 5-8 lines]
My DevOps journey started [your origin story — a production incident, a curiosity about automation, a college project]. Since then, I've [2-3 key career highlights].

I specialize in:
→ [Domain 1]: [1-line detail]
→ [Domain 2]: [1-line detail]
→ [Domain 3]: [1-line detail]

[CTA — 2-3 lines]
[What you want — hiring, connecting, collaborating]. Let's connect if you're [target audience action].

📩 your.email@gmail.com
🔗 github.com/yourusername

#DevOps #Kubernetes #AWS #CloudEngineering #[YourCity]
```

### Fresher Example

```
I build CI/CD pipelines and containerized applications for fun — and I'm ready to do it professionally. Through 5 personal projects and a 6-month internship, I've automated deployments, provisioned cloud infrastructure, and broken (then fixed) more things than I can count.

My DevOps journey started when I deployed my first Docker container during college and thought "wait, THIS is how software gets shipped?" That moment changed everything. Since then, I've earned my AWS Cloud Practitioner certification, built a full GitOps pipeline on Minikube, and started contributing to open source.

I specialize in:
→ CI/CD: GitHub Actions, Jenkins
→ Containers: Docker, Kubernetes (Kind/Minikube)
→ Cloud: AWS (EC2, S3, IAM, VPC)

Looking for DevOps / Cloud Engineer roles where I can learn from experienced engineers and contribute to real infrastructure challenges. Open to Pune, Bangalore, or remote.

📩 yourname@gmail.com
🔗 github.com/yourusername

#DevOps #AWS #Kubernetes #Docker #OpenToWork
```

### Experienced Example

```
I help engineering teams ship faster without breaking things. With 5+ years in DevOps and SRE, I've migrated 30+ microservices to Kubernetes, reduced deployment failures by 80%, and cut AWS costs by $40K/month across three companies.

My career started in a service company managing servers manually. The first time I automated a 4-hour manual process into a 10-minute pipeline, I was hooked. Since then, I've gone deep into platform engineering, building internal developer platforms that let dev teams self-service their infrastructure.

I specialize in:
→ Container Orchestration: Kubernetes (EKS/GKE), Helm, ArgoCD, Istio
→ Infrastructure as Code: Terraform (200+ modules), Ansible, CloudFormation
→ Observability: Prometheus, Grafana, ELK, OpenTelemetry

Open to Senior DevOps / Platform Engineering / SRE roles (remote or Pune-based). Let's connect if you're building resilient infrastructure or want to chat about K8s at scale.

📩 yourname@gmail.com
🔗 github.com/yourusername | yourblog.dev

#DevOps #Kubernetes #AWS #PlatformEngineering #SRE
```

## Post Templates (Quick Content Ideas)

### TIL Post (5 min to write)
```
TIL that [tool/command/concept] can [what it does].

[2-3 lines explaining the context]

Here's the command:
[code snippet or screenshot]

I've been doing DevOps for [X years] and just found this.
What's something obvious you learned embarrassingly late?

#DevOps #TIL #[Tool]
```

3. [Learning 3]

## Connection Request Templates (300 chars max)

**After a meetup/event:**
"Hi [Name], great meeting you at [event]. Loved the discussion on [topic]. Would love to stay connected and continue the conversation."

**To a recruiter:**
"Hi [Name], I'm a DevOps engineer with [X] years in [K8s/AWS/Terraform]. I see you recruit for [company/domain]. Would love to connect in case there's a fit."

**To a peer:**
"Hi [Name], I saw your post about [topic] — really insightful. I work in a similar space ([your tools]). Would love to connect and exchange ideas."

**Cold outreach to hiring manager:**
"Hi [Name], I'm impressed by [company]'s work on [specific thing]. I'm a DevOps engineer with experience in [2-3 tools]. Would love to learn more about your team."

## Quick Reference Numbers

| Metric | Target |
|--------|--------|
| SSI Score | 70+ (check at linkedin.com/sales/ssi) |
| Headline Length | Use all 120 characters |
| About Section | Use 1500+ of 2600 characters |
| Skills Listed | 20-30 (not 5, not 50) |
| Top 3 Skills Endorsements | 10+ each |
| Recommendations | 3-5 minimum |
| Featured Items | 3-4 pinned |
| Posting Frequency | 1x/week minimum |
| Connection Requests | 10-15/week with notes |
| Comments on Others' Posts | 5+/week |
