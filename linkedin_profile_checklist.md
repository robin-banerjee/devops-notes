# LinkedIn Profile Checklist for DevOps Engineers
## By TrainWithShubham | LinkedIn Masterclass

Use this checklist to audit your profile. Every unchecked box is a missed opportunity.

---

## Profile Photo & Banner

- [ ] Professional headshot uploaded (face = 60% of frame)
- [ ] Good lighting, neutral background
- [ ] Recent photo (within last 2 years)
- [ ] Custom banner uploaded (1584 x 396 px)
- [ ] Banner includes: tech stack, title, or personal brand tagline
- [ ] Banner has website URL or GitHub link visible

## Headline (120 characters)

- [ ] NOT just "Job Title at Company"
- [ ] Includes 3-4 searchable keywords (e.g., Kubernetes, AWS, Terraform)
- [ ] Follows the formula: [Role] | [Skill 1] | [Skill 2] | [Skill 3] | [Differentiator]
- [ ] Readable and not keyword-stuffed

## About Section (2600 characters max)

- [ ] First 3 lines are a strong hook (shows above "see more")
- [ ] Paragraph 1: What you do + biggest achievement
- [ ] Paragraph 2: Your journey / story / what drives you
- [ ] Paragraph 3: CTA — what you want people to do (hire, connect, collaborate)
- [ ] Written in first person ("I build..." not "He/She is...")
- [ ] Contains relevant keywords naturally
- [ ] Uses line breaks and short paragraphs (not a wall of text)
- [ ] Includes email/contact info at the bottom
- [ ] Has a hashtag line for searchability

## Experience

- [ ] All roles listed with accurate dates and company names
- [ ] Job titles match your resume (consistency!)
- [ ] Each role has 3-5 bullet points with context and impact
- [ ] Bullets expanded beyond resume (stories, not just metrics)
- [ ] Media attached (architecture diagrams, demos, blog links)
- [ ] Companies tagged with @ mentions
- [ ] Most recent role has the most detail

## Featured Section

- [ ] Section is NOT empty
- [ ] At least 3-4 items pinned
- [ ] Includes: GitHub repo with clean README
- [ ] Includes: Blog post or technical article
- [ ] Includes: Certification badge (Credly/AWS/CKA)
- [ ] Includes: Demo video or conference talk (if available)
- [ ] Items ordered by strength (strongest first)

## Skills & Endorsements

- [ ] Top 3 skills are strategically chosen (match target role keywords)
- [ ] 20-30 skills listed (not 5, not 50)
- [ ] Irrelevant skills removed (MS Word, PowerPoint, etc.)
- [ ] Skills categorized mentally: Containers, CI/CD, Cloud, IaC, Monitoring, Scripting
- [ ] At least 10+ endorsements on top 3 skills
- [ ] You've endorsed colleagues to encourage reciprocation

## Recommendations

- [ ] At least 2-3 recommendations received
- [ ] Recommendations are from managers, leads, or senior colleagues
- [ ] Recommendations mention specific projects or skills
- [ ] You've given recommendations to others (reciprocity)

## Education & Certifications

- [ ] Education section complete with degree, university, year
- [ ] Relevant certifications added (AWS, CKA, Terraform, etc.)
- [ ] Credly badges linked for visual badges on profile
- [ ] Expiration dates updated (no expired certs showing)
- [ ] Relevant online courses listed (TWS, KodeKloud, etc.)

## Activity & Engagement

- [ ] Posted at least 1 post in the last 30 days
- [ ] Commented on 5+ posts in your domain this week
- [ ] Joined 3-5 relevant groups (CNCF, AWS UG, DevOps communities)
- [ ] Profile set to "Public" (not private)
- [ ] SSI score checked (target: 70+) at linkedin.com/sales/ssi

## Open to Work Settings

- [ ] "Open to Work" enabled (Private for employed, Public for job seekers)
- [ ] Specific job titles set (not just "DevOps")
- [ ] Location preferences configured
- [ ] Remote/on-site preference set
- [ ] Job types selected (Full-time, Contract, etc.)

## Consistency Check

- [ ] Job titles match across Resume, LinkedIn, and GitHub
- [ ] Dates match across all platforms
- [ ] Company names are consistent
- [ ] Skills mentioned on resume appear in LinkedIn headline/skills
- [ ] GitHub repos match projects mentioned in Experience/Featured

---

## Scoring

Count your checked boxes:

| Score | Level | Action Needed |
|-------|-------|---------------|
| 0-15 | Needs Work | Start with Photo + Headline + About |
| 16-30 | Getting There | Fill out Featured + Skills + Experience details |
| 31-40 | Strong | Add recommendations, start posting content |
| 41-50 | All-Star | Maintain with weekly activity + content |
